## 🚀 Key Features
 1. **Ultra-Sleek Developer Portal**: Dark future-proof UI utilizing tailored Tailwind CSS gradients and glow effects.
 2. **Interactive Handshake Simulator**: A real-time visual package mapper showing how SSL-encrypted JSON-RPC frames transit from LLM clients down to local nodes.
 3. **Advanced Multi-Agent Playground**: Interactive sandbox allowing users to query local nodes in natural language and observe simulated tool routing and raw JSON payloads.
 4. **Live Stream Telemetry**: Comprehensive logs panel recording handshake events, latencies, request routes, and success rates.
 5. **Minimal Domain Sales Banner**: Integrated, non-intrusive domain-for-sale CTA with an overlay email submission form.

## 💻 Local Setup & Run
Since the entire system is built following the **Single-File Mandate**, you can run this instantly inside any standard React application environment.
 1. Ensure you have Node.js installed.
 2. Clone or copy BeamMCP_App.jsx into your React directory (rename to App.jsx if necessary).
 3. Ensure Tailwind CSS is configured in your project:
   ```bash
   npm install -D tailwindcss postcss autoprefixer
   npx tailwindcss init
   
   ```
 4. Run the development server:
   ```bash
   npm run dev
   
   ```
 5. Open your local port to test the live simulator!
